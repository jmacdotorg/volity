Volity-WebClient

This is the server-side core of the official Volity web client,
implemented in Perl.

It is very PRE-ALPHA. No part of its internals or its API should be
considered complete or even functional, and all parts are subject to
violent change.

INSTALLATION

To install this module, run the following commands:

    perl Makefile.PL
    make
    make test
    make install


SUPPORT AND DOCUMENTATION

After installing, you can find documentation for this module with the perldoc command.

    perldoc Volity::WebClient

Basic information about the whole web client project is visible at the
Volity wiki:

    http://www.volity.org/wiki/index.cgi?Web_Client
    
For more information about Volity in general, please visit http//volity.net.
