Message Localization

Some automated Volity entities, such as the bookkeeper (during [game scheduling]?) and bots, might occasionally want to send plain Jabber messages to people. Jabber has great localization support, making it easy for such a program to send out several language-tagged versions of the same message and have the correct one display for the recipient.

This page exists to let the Volity community provide translations for these messages. (To help translate message tokens specific to Gamut, see Gamut localization.)

How to Help

To add a translation, simply write it directly beneath the original statement, adding a key that is its two-letter language code.

Example:

en
Hello!
es
¡Hola!
fr
Bonjour!
Do not contribute a translation unless you are fluent in that language and are positive that your statement would sound natural to a native speaker. Anyone can use Google's language tools, but machine-translation's results on all but the simplest statements are famously funny-sounding. The goal here is to let Volity-based programs sound natural.

Feel free to add additional sentences that require translation.
