The big question

How close are we to declaring a feature freeze on Volity 1.0? Doing so is a prerequisite for jmac undertaking the authorship of the platform spec.

Note that the "features" referred to here don't need to have been implemented by anyone yet; they merely have to be described well enough to allow someone to implement them.

Supported features

Low-level

Transport and basic communication.
  • User account creation and management (XMPP)
  • User connection and authentication (XMPP)
  • Communication between users (XMPP)

Mid-level

Gameplay.
  • Table creation
  • Table configuration
  • Invitations
  • Game play API (Referee)
  • Standard game record format

High-level

Meta-gameplay.
  • Game Finder
  • UI Finder
  • Lobbies
  • Reputation System
