Volityd

HomePage | RecentChanges | Preferences | Upload
Volityd (the Volity Daemon, you see) is a full-featured Volity parlor program, though it offers a text-only, command-line interface. Like the Frivolity package that it accompanies, it is written in [Perl], primarily by Jason McIntosh.

As its name implies, it has a user interface typical of Unix network-service daemons: it's meant to be run as a background process, and offers no interactivity once it has started. All configuration (such as specifying which game module to use) occurs on launch through command-line switches. In the future, we hope to have it support configuration files as well.

Documentation

Volityd contains embedded documentation in Perl's POD format; you can use the perldoc command to read it, or man after it's been installed. You can also [view the docs as HTML].
