Volity.Net Game Finder

The volity.net game finder is volity.net's official implementation of the game finder. It runs as a Web-based application separate from any Volity client application. Its Web interface, hosted on volity.net and usable with any reasonably modern Web browser, will allow users to browse the Volity game world, starting with a broad view of available rulesets and giving them the ability to drill down all the way to the table level. When a user finds a table, parlor or lobby they wish to join, the game finder will upload a volity file to them. The user's operating system will (through MIME or file-extension magic) pass this along to Volity client, which will then do the right thing.

We hope to make additional views available so that clients such as Gamut can display the finder within its own windows, rather than redirecting users to their respective Web browsers.

A basic form of the Game Finder is available. It will become more functional over time.
