WhyThisIsCool

This page will hold a list of PR-appropriate reasons why the MIGS/Volity? game system is cool and unique. Please add to it as you think of things.

Update, 12/29/2003: I've added something like is as a section to the VolityBook.

User-level reasons (a.k.a. Player-level reasons)

  • Download a single client application for your computer, and you can play all MIGS games! (Compliant ones, anyway.)

  • Massively multiplatform, since the servers do all the work. Use a Web-based client if all else fails.

  • i18n and l9n support (with game servers that support localized strings & such)

Game-designer (module-writer) reasons

  • Game module authors won't have to themselves run or even have access to a Jabber server in order to run a Volity game server, since a game server can run as a simple user of any existing Jabber server with which the author can acquire a Jabber login. Resource leveraging is cool.

  • You can make computer games without worrying about user management, user communication, state-saving, or network protocols. or creating client applications. A lot of UI work is done for you, but you still need to design a game-specific UI.
