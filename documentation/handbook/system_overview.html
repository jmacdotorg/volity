Volity concept index

Jabber entities

The Volity network entirely comprises only a few different types of entities, all of which are simply connections to the underlying Jabber network.

  • parlor
  • referee
  • player
  • table
  • bookkeeper
  • client (Not really a component per se, but the software that enables human players to become parts of the Volity machine.)
