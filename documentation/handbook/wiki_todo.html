Detailed game story

In-depth example of the lifetime of a single table and referee, telling the tale of:

  • A Volity user using the game finder to locate a parlor
  • Creation of a new table
  • Invitation of other users
  • Still other users wandering in
  • Bots summoned
  • Users and the referee firing off readiness calls, and handling the ones they receive
  • The users organizing into players (some multi-user, some remaining single-user)
  • Starting the game
  • Playing the game (this can be summarized)
  • Ending the game
  • The referee creating the game record and sending it to the bookkeeper
  • The users leave, and the referee cleans itself up and shuts down
