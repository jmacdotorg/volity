Volity.Net

The volity.net website

The website running at http://volity.net is going to serve several purposes:

  • The enduser-friendly Web front-end to the public Volity network, and an introduction into the world of Volity.
  • A place for Volity players to look up game histories, browse player stats, and maintain their personal records.
  • Game developers' interface to the volity.net bookkeeper, giving them an opportunity to register and maintain their parlors, UI files, and rulesets.

As of March 2006 it's under rapid development, mostly by [You can visit its current introductory page] and log in with your volity.net ID. (If you don't have a volity.net ID, you can create one with Gamut.)

The volity.net Jabber server

There is also a Jabber server running at volity.net, maintained by Andy Turner. It hosts all of Volity's Jabber entities that the core Volity team runs, such as the bookkeeper and the test servers.

Since it allows open user regsitration, any user can create Jabber accounts there, which can be applied to Volity players, ronin, or parlors.
