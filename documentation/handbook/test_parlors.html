Test Parlors

Volity test parlors are parlors (game servers) created and maintained by Volity's core development team (well, by Jmac, at least). They are meant to serve as stable platforms to test client applications against, and provide model examples for Volity game authors through their open source code (freely available through Volity SVN).

Here are the current servers, and their JIDs:

Barsoomite Go
zarf-barsoom@volity.net/volity (There is a four-player version at zarf-barsoom4@volity.net/volity)
Aquarius
zarf-aquarius@volity.net/volity
Rock, Paper, Scissors
rps@volity.net/volity This is currently down, pending being brought up to date with current spec.
Crazy Eights
eights@volity.net/volity This is currently down.
