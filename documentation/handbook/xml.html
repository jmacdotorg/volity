XML is a way to encode hierarchical document information in files.

Volity uses lots and lots of mechanisms which are based on XML. These include:

  • SVG
  • Jabber
  • Jabber-RPC, which is based on XML-RPC
  • Localization files in UI bundles
  • Pluggable resources
