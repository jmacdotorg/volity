Multi-User Chat

Multi-user chat, or MUC, is a Jabber extension protocol that defines chatrooms. It is sufficiently rich and flexible for Volity to use as a basis for its tables; a Volity table, in fact, is simply a MUC instance with an attached referee. It's the latter's presence that elevates it from an IM chatroom to a game table.
