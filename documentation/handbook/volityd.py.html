Volityd.Py

volityd.py is a Python implementation of the Volity Parlor, Referee, and game module programming framework.

(Unlike Frivolity, volityd.py does not include a bookkeeper implementation, nor a friv-style text client.)

You can download volityd.py (and the Python modules on which it depends) from the SVN repository on Volity's SourceForge site.

Status

Matches current Volity specification.
