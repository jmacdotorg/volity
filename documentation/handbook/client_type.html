A Volity client type is a category of client applications, labeled by a URI. Every client falls under a client type.

The bookkeeper keeps its own registry of all known client types, so that when a client seeks a UI file through the UI finder, the bookkeeper can properly direct it to an appropriate file.

List of known client types

http://volity.org/protocol/ui/text
A basic text client, such as Friv.
http://volity.org/protocol/ui/svg
A basic SVG client, such as Gamut.
http://volity.org/protocol/ui/html
A prospective (read: fantasy) type for an in-browser client under development.

Other possibilities: a play by email client type.
