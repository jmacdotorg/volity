Game

We bat the word game around a lot when talking about Volity, since games are Volity's reason for being. However, when talking system design, the word often has a more precise meaning: a single instance of a game in play. In other words, the thing that happens at a Volity table. This is distinct from the concept of the game as a collection of rules for play, which we call rulesets, or the entities on the network you connect to in order to play games -- the parlors.

Mind your context and all will be well.
