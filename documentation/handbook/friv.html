Friv

Friv is a Volity client program, written during the early days of the project. However, it offers a text-only, command-line interface (no graphics or even text formatting) and it is missing many features. Once part of the Frivolity package, it is now no longer maintained, since any purpose it served has been superceded (and then some) by Gamut.

Like the Frivolity package that it accompanies, it is written in Jason McIntosh (after an initial nudge by Andy Turner).

Download

You can find the final Friv release as part of http://prdownloads.sourceforge.net/volity/Frivolity-0.5.2.tar.gz?download

As of version 0.6.0, Friv is no longer packaged with Frivolity.

Documentation

Friv contains embedded documentation in Perl's POD format; you can use the perldoc command to read it, or man after it's been installed. You can also [view the docs as HTML].
