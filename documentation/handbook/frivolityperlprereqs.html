FrivolityPerlPrereqs

There will be a apt-source (both sid and woody) for .deb's of Perl modules that Frivolity needs that are not currently in Debian.

There is a copy of the source for each required module available [here].

Someday this list will be broken down by which Frivolity compent requires which bit.
