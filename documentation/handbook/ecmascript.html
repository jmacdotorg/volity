ECMAScript

ECMAScript is the proper name of the programming language popularly known as JavaScript, and the lingua franca of Volity UI Files.

Both extant Volity client programs -- Friv and Gamut -- have built-in ECMAScript interpreters capable of comprehending and executing UI files, including those obtained through the UI Finder.
